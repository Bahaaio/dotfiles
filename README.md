## Installation

Uses [install.sh](install.sh) to install dotfiles.

```sh
curl -fsSL https://raw.githubusercontent.com/Bahaaio/dotfiles/main/install.sh | bash
```

> [!NOTE]
> Use [link.sh](link.sh) to symlink dotfiles manually.
