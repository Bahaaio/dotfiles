# WM Role

Installs desktop environments and window managers.

## What it does

- **i3**: Tiling window manager with polybar, picom, dunst, and utilities
- **GNOME**: Full GNOME Desktop with extensions and tweaks
- **Launchers**: rofi and ulauncher application launchers

Both i3 and GNOME are installed, allowing you to choose at login (GDM3).
