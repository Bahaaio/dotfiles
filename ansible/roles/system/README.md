# System Role

Installs base system packages and configures system services.

## What it does

- Installs essential system utilities (curl, wget, git, build-essential, etc.)
- Installs and configures TLP for laptop power management
- Installs and configures keyd for keyboard remapping (capslock/escape swap)
- Installs and configures ufw firewall (deny incoming, allow outgoing, localsend ports)
- Updates apt cache
